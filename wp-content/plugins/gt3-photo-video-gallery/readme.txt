== Description ==

This powerful and free photo and video gallery WordPress plugin allows you to extend the functionality of the default WordPress gallery. You can easily customize the look and feel of the photo or video gallery using custom GT3 gallery settings panel. Create a new look for your awesome galleries with ease.

== Installation ==

After downloading the ZIP file,

1. Log in to WordPress dashboard panel.
2. Go to Plugins Add > New > Upload.
3. Click Choose file (Browse) and select the downloaded zip file of the gallery.
4. Click Install Now button.
5. Click Activate Plugin button for activating the gallery plugin.
If the installation does not succeed, please contact us.
